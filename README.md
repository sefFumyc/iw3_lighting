# iw3_lighting
a mod to add light in iw3

## How to use
Use **Extreme Injector** or any Injector to inject dll to your **iw3mp.exe** or **iw3xo.exe**

Press **Insert** key to toggle UI

it can work in DX9 or DX11

it can work with iwxmvm(only DX9)
